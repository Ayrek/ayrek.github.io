Absolute Positioning

There are seven important points of the absolute position type:

  • Lets you place any page element wherever you want.
  • The value is relative to the positioned parent element not the viewport.
  • And if there isnt a positioned parent element then the defeault is the html page. This could cause scrolling with the page.
  • The element is removed from the flow on the page.
  • Not affected by other elements.
  • Other elements are not affected by the element
  • Potentially this could limit the flexibility of the web page.
